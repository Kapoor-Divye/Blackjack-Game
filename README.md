# Blackjack-Game
A simple browser-based Blackjack game built using HTML, CSS, and JavaScript. This project focuses on core JavaScript concepts like arrays, random number generation, and game logic while keeping the UI minimal and intuitive.

📌 Description

This is a basic implementation of the classic card game Blackjack (21) where the goal is to get as close to 21 as possible without going over.

The player can:

Draw cards
Track their total sum
Win, lose, or continue based on game rules

It’s a great beginner-friendly project to understand how real-world logic can be translated into code.

🚀 Features
🎴 Random card generation
➕ Draw new cards
📊 Live sum calculation
🎯 Win/Lose conditions
🧠 Simple game state management
⚡ Lightweight and fast
